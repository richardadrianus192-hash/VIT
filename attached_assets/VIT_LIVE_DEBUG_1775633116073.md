## Issue 1: All Predictions Have 0.00% Edge

**What's happening:**
```
Screenshot shows:
  Edge: 0.00%
  Stake: 0.00%
  Prediction: NONE
```

**Why this is wrong:**
- Edge should vary by match (5.2%, -1.3%, 3.8%, etc.)
- 0.00% edge means model prediction = market odds exactly
- This indicates **fallback is still being used** (converting odds, not using models)

**Root cause:**
Even though 11/12 models are loaded, they're probably not actually running or returning garbage values.

## Issue 2: "Prediction: NONE" Always

**What's happening:**
- Best side always shows "NONE" (no bet recommendation)
- Should show "HOME", "DRAW", or "AWAY"

**Why:**
The Kelly Criterion calculation is returning zero edge, so no side is recommended.

**Code location:**
```python
# app/api/routes/predict.py
best_bet = MarketUtils.determine_best_bet(
    home_prob, draw_prob, away_prob,
    home_odds, draw_odds, away_odds
)
# ← This returns edge = 0.00
```

**Why edge is zero:**
- Model probabilities = market implied probabilities
- Difference = 0
- Edge = 0

**This means:** Models aren't predicting differently from the market.

## Root Cause Analysis

### The Core Problem

Even though 11/12 models load, **they're not actually being used in predictions**.

**Flow:**
```
1. orchestrator.predict() is called
   ↓
2. Try to run 11 models
   ↓
3. Models either:
   a) Fail to run (exception caught)
   b) Don't set supported_markets correctly
   c) Return invalid probabilities
   d) Are filtered out for some reason
   ↓
4. After line 171, no models contributed
   ↓
5. orchestrator falls back to: _hybrid_fallback()
   ↓
6. Returns: Convert market odds → probability
   ↓
7. Frontend shows: Generic prediction, 0% edge
```

```javascript
export function ModelDebug({ prediction }) {
  if (!prediction?.individual_predictions) return null;
  
  const successful = prediction.individual_predictions.filter(
    p => p.status === 'success'
  );
  
  return (
    <div style={{ 
      padding: '12px', 
      backgroundColor: '#f8f9fa', 
      borderRadius: '6px',
      marginTop: '12px'
    }}>
      <h4>Model Breakdown ({successful.length}/{prediction.individual_predictions.length})</h4>
      <table style={{ width: '100%', fontSize: '12px' }}>
        <thead>
          <tr style={{ borderBottom: '1px solid #ddd' }}>
            <th style={{ textAlign: 'left', padding: '4px' }}>Model</th>
            <th style={{ textAlign: 'center', padding: '4px' }}>Home %</th>
            <th style={{ textAlign: 'center', padding: '4px' }}>Conf</th>
            <th style={{ textAlign: 'center', padding: '4px' }}>Status</th>
          </tr>
        </thead>
        <tbody>
          {prediction.individual_predictions.map((p, idx) => (
            <tr key={idx} style={{ borderBottom: '1px solid #eee' }}>
              <td style={{ padding: '4px' }}>{p.model_name}</td>
              <td style={{ textAlign: 'center', padding: '4px' }}>
                {(p.home_prob * 100).toFixed(0)}%
              </td>
              <td style={{ textAlign: 'center', padding: '4px' }}>
                {(p.confidence?.['1x2'] || 0.5).toFixed(2)}
              </td>
              <td style={{ 
                textAlign: 'center', 
                padding: '4px',
                color: p.status === 'success' ? '#28a745' : '#dc3545'
              }}>
                {p.status === 'success' ? '✓' : '✗'}
              </td>
            </tr>
          ))}
        </tbody>
      </table>
    </div>
  );
}
```
